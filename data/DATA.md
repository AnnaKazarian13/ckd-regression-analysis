# Data

## File
`kidney_dataset.csv`

## Shape
5000 rows × 11 columns

## Columns
- Creatinine, BUN, GFR, Urine_Output
- Diabetes, Hypertension, Age
- Protein_in_Urine, Water_Intake, Medication
- CKD_Status

## Notes
- Tabular kidney / GFR dataset used for educational regression analysis.
- `CKD_Status` is kept in the file for EDA / leakage demonstration, but **not used** as a feature in the final model (CKD is typically defined via GFR).
- If this dataset is synthetic or from a course pack, treat results as methodological demo, not clinical validation.
